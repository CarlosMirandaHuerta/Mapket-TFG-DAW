# Roadmap de desarrollo

## Fase 0: Planificación

Objetivo:

- Definir producto.
- Definir arquitectura inicial.
- Definir modelo de datos.
- Definir fases de desarrollo.

## Fase 1: Setup de Vue

Objetivo:

- Crear app Vue 3 con Vite y JavaScript.
- Configurar estructura base.
- Crear rutas iniciales.
- Preparar estilos globales mobile-first.


## Fase 2: Mapa demo interactivo

Objetivo:

- Crear mapa visual de una tienda demo.
- Dividir el supermercado en secciones con colores y títulos.
- Permitir seleccionar secciones.
- Mostrar productos o categorías por sección.


## Fase 3: Búsqueda, filtros y QR

Objetivo:

- Crear datos mock de productos.
- Buscar por nombre, categoría y marca.
- Filtrar por categoría.
- Leer `from` desde la URL.
- Marcar "estás aquí" en el mapa.
- Resaltar la ubicación del producto seleccionado.


## Fase 4: Base de datos

Objetivo:

- Crear esquema SQL inicial.
- Preparar datos seed de tienda demo.
- Conectar Vue con Supabase.
- Reemplazar mocks por datos reales.


## Fase 5: Panel de empleado

Objetivo:

- Login de empleado.
- Lista de productos.
- Edición de ubicación.
- Edición de cantidad y estado de stock.
- Historial básico de cambios.


## Fase 6: Presentación comercial


Objetivo:

- Pulir UI y experiencia móvil.
- Crear tienda demo convincente.
- Preparar textos de presentación.
- Generar QR de prueba.
- Revisar rendimiento y accesibilidad básica.


## Fase 7: Despliegue y base de datos real

Objetivo:

- Preparar despliegue público de la app.
- Configurar redirecciones para Vue Router en producción.
- Documentar variables de entorno de Supabase.
- Documentar la puesta en marcha de la base de datos real.
- Dejar claro que clientes y empleados solo usan una URL pública.



## Criterios generales de cada fase

- Cada fase debe resolver un bloque claro.
- Mantener la app usable en móvil desde el principio.
